Interactive Elements

In-text link

CSS selector:

.link

Sample code:

<p>this is normal p<a href="www.google.com" class="link">click here for an in-text link </a> this is normal p</p>

Rendered element:

start of p click here for an in-text link end of p

Navigation Link

CSS selector:

.button-nav-container .button-nav

Sample code:

                    <a class="button-nav" href="#">
                        Link To Section
                    </a>

Rendered element:

Link To Section

Radio Button with Label

CSS selector:

.radio-input .radio-label

Sample code:

                    <div class="radio-input">
                        <input type="radio" name="example-radio" id="radio-element">
                        <label class="radio-label" for="radio-element"> text here </label>
                    </div>
